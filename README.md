# SwiftyChevron

SwiftyChevron is a dynamic iOS framework which allows you to use a fully Swift written dynamic chevron for you apps.

## Install

### Cocoapods

```ruby
pod 'SwiftyChevron'
```

### Carthage

```ruby
github "iMac0de/SwiftyChevron"
```
